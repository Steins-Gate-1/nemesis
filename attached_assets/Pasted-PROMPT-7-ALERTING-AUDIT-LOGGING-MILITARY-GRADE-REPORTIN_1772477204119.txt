IF breach_count > 3
→ Create HIGH alert

IF critical CVE detected
→ Create HIGH alert

IF honeytoken triggered
→ Create CRITICAL alert

IF deepfake probability > 85%
→ Create CRITICAL alert

IF risk_score crosses threshold
→ Escalate severity

• id (UUID)
• alert_type
• domain
• severity_level
• source_module
• related_object_id
• description
• recommended_action
• status (OPEN / ACKNOWLEDGED / RESOLVED)
• created_at
• updated_at

If:

Multiple alerts within 10 minutes
→ Escalate severity level

Active targeting + high exposure
