🛡️ 1️⃣ HAVE I BEEN PWNED (Breach Detection)
🔗 https://haveibeenpwned.com/API/v3

🪜 Step-by-Step:
Step 1 — Create Account
* Go to website
* Purchase API key (cheap yearly subscription)
* Copy API key

Step 2 — Install Python Requests

pip install requests


Step 3 — Make API Call

import requests

api_key = "YOUR_API_KEY"
email = "test@example.com"

headers = {
    "hibp-api-key": api_key,
    "user-agent": "NEMESIS"
}

url = f"https://haveibeenpwned.com/api/v3/breachedaccount/{email}"

response = requests.get(url, headers=headers)

if response.status_code == 200:
    breaches = response.json()
    print(breaches)
else:
    print("No breaches or error:", response.status_code)


What You Store in DB:
* Breach name
* Date
* Data types exposed
* Severity score

🕸️ 5️⃣ NEO4J GRAPH DATABASE
🔗 https://neo4j.com/download/

🪜 Steps:
Step 1 — Install Neo4j Desktop
Step 2 — Install Python Driver

pip install neo4j


Step 3 — Connect & Create Nodes

from neo4j import GraphDatabase

driver = GraphDatabase.driver("bolt://localhost:7687", auth=("neo4j", "password"))

def create_email(tx, email):
    tx.run("CREATE (e:Email {address: $email})", email=email)

with driver.session() as session:
    session.write_transaction(create_email, "test@example.com")


Step 4 — Create Relationships

tx.run("""
MATCH (e:Email {address:$email})
MATCH (d:Domain {name:$domain})
CREATE (e)-[:BELONGS_TO]->(d)
""", email=email, domain=domain)
